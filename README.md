## 功能特点

### 新闻爬虫
- CCTV爬虫：异步多线程爬取CCTV网站搜索结果，支持自动分页和批量处理
- 人民网爬虫：支持同步/异步模式，提取完整文章内容，可按时间范围筛选

### 电商爬虫
- 京东商品爬虫：爬取京东商品信息、评论数据和价格信息

### 应用商店爬虫
- Google Play评论爬虫：使用Playwright自动化浏览器爬取应用评论，支持中英文界面
